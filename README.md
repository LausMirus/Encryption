## Shell script for password encryption

---

***This Bash script is an interactive command-line tool used to generate cryptographic hash values for a user-provided string. It supports multiple hashing algorithms and allows the user to choose which algorithm to apply at runtime.***

___

___The script is useful for:___
* Learning basic Bash scripting concepts.
* Verifying data integrity.
* Generating hashes for passwords, tokens, or 
* identifiers (for educational purposes).

___

___Features___

* Takes user input dynamically.
* Supports multiple hashing algorithms:
1) MD5
2) SHA-256
3) SHA-512
* Uses standard Linux utilities (md5sum, sha256sum, sha512sum).
* Simple and lightweight, no external dependencies.
